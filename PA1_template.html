3. Create a new dataset that is equal to the original dataset but with the missing data filled in.

First we create a dataset called df_na that contains only the row of df where the steps are missing. We check the number of missing values and the number of rows in the dataset to make sure theyre in line with the original dataset.

df_na<-df[is.na(df$steps),]
colSums(is.na(df_na))
##    steps     date interval

nrow(df_na)
## [1] 2304

Then we are going to pull data from byinterval which is the dataset showing the average number of steps by interval.

For each row in df_na, were going to look for the steps value corresponding to the given interval in the byinterval dataset.

Then we check that theres no null value anymore and that we still have the right number of rows.

for (i in (1:nrow(df_na))) {
    df_na[i,1]=byinterval[which(byinterval$interval==df_na[i,3]),2]
}
